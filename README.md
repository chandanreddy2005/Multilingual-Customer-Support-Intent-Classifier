# Multilingual Customer Support Intent Classifier

An AI/ML Streamlit application that classifies customer support messages into seven intent categories across English and four Indian-language transliteration sets: Hindi, Kannada, Telugu, and Tamil.

## Features

- Real-time customer intent prediction
- Confidence score and per-intent probability chart
- Language detection with `langdetect`
- Balanced generated dataset with 1,575 samples
- TF-IDF word and character n-gram features
- Production model uses calibrated Linear SVM with hybrid word and character TF-IDF
- Logistic Regression and Naive Bayes comparison
- Analytics dashboard with confusion matrix, intent distribution, model comparison, and language-wise accuracy

## Supported Intents

| Intent | Example |
|---|---|
| Billing Dispute | "I was charged twice" |
| Refund Request | "I want my money back" |
| Delivery Inquiry | "Where is my order?" |
| Account Recovery | "I forgot my password" |
| Product Defect | "The item is broken" |
| Technical Support | "App keeps crashing" |
| Payment Failure | "My payment failed" |

## Quick Start

```bash
pip install -r requirements.txt
python train.py
python validate_project.py
streamlit run app.py
```

If the app opens before training artifacts exist, it will show a "Train model now" button.

## Training Output

`python train.py` regenerates:

- `data/customer_support_data.csv`
- all files in `saved_models/`
- all charts in `visuals/`

The app loads `saved_models/model_bundle.pkl` first. It also supports the legacy pickle files so older generated artifacts do not immediately break the UI.

## Validation

Run:

```bash
python validate_project.py
```

This checks that the core files, model artifacts, visuals, and model bundle keys are present.

## Deployment

### Streamlit Cloud

1. Push this repository to GitHub.
2. Set the app entry point to `app.py`.
3. Make sure `requirements.txt` is included.
4. Either commit trained artifacts or run `python train.py` during setup.

### Render

Build command:

```bash
pip install -r requirements.txt && python train.py
```

Start command:

```bash
streamlit run app.py --server.port $PORT --server.address 0.0.0.0
```
